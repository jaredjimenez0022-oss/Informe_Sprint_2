\chapter{Conclusiones y trabajo futuro}

\section{Conclusiones}

Durante el \textbf{Sprint 2} del proyecto \textbf{Komuness} se logró implementar un sistema completo de \textbf{monetización y membresías premium}, estableciendo una base económica sólida sin perder el enfoque comunitario. Los principales avances fueron:

\begin{itemize}
    \item \textbf{Membresías funcionales:} Diferenciación entre usuarios básicos (10 publicaciones) y premium (50), con configuración dinámica y planes accesibles.
    \item \textbf{Integración con PayPal:} Implementación completa con verificación segura, manejo de webhooks y prevención de cobros duplicados.
    \item \textbf{Gestión de errores robusta:} Sistema de reintentos que recuperó el 87\% de pagos fallidos, con mensajes claros para el usuario.
    \item \textbf{Moderación avanzada:} Ediciones controladas con vista previa antes/después y flujo de aprobación que mantiene la calidad del contenido.
    \item \textbf{Mejor experiencia de usuario:} Alertas preventivas, barra de progreso y botón de mejora directa.
    \item \textbf{Corrección de deuda técnica:} Mejoras en búsquedas, validaciones de archivos y recuperación de contraseñas.
    \item \textbf{Arquitectura escalable:} Código modular en TypeScript con documentación clara.
    \item \textbf{Mayor eficiencia del equipo:} Reducción de sobrecarga y aumento de velocidad (21→33 puntos).
\end{itemize}

\section{Problemáticas y limitaciones}

\begin{itemize}
    \item Uso actual de \textbf{PayPal Sandbox}, pendiente migración a producción.
    \item Falta de \textbf{notificaciones para usuarios} (pagos y ediciones).
    \item Ausencia de \textbf{dashboard de métricas} y análisis de conversión.
    \item \textbf{Ediciones limitadas} sin reinicio y membresías vencidas sin automatización.
    \item Solo \textbf{PayPal} como método de pago; falta soporte para Stripe o SINPE.
    \item Planes \textbf{fijos} y sin suscripciones automáticas.
    \item Sin historial visible de pagos ni priorización en revisión de ediciones.
\end{itemize}

\section{Trabajo futuro}

\begin{itemize}
    \item \textbf{Migrar a producción} e integrar \textbf{Stripe} y \textbf{SINPE Móvil}.
    \item Implementar \textbf{notificaciones multicanal} (email y WhatsApp).
    \item Crear \textbf{dashboard de analíticas} con métricas de ingresos, conversiones y engagement.
    \item Incorporar \textbf{suscripciones recurrentes} mediante PayPal Subscriptions API.
    \item Desarrollar \textbf{cupones y promociones} para campañas de marketing.
    \item Mejorar \textbf{moderación} con filtros, comentarios y reinicio de créditos de edición.
\end{itemize}
